# Rover_1 - Autonomous Waypoint Navigation

This repository contains the source code for the promotional website for Rover_1, a compact autonomous rover designed for smooth, precise waypoint navigation.

## Overview

Rover_1 is a fully autonomous ground vehicle designed to navigate between GPS waypoints with precision and stability. Built from the ground up with modular, Python-based control software, it integrates IMU fusion, mecanum kinematics, and predictive motion control to deliver smooth, adaptive movement.

This website serves as a showcase for the project's features, build specifications, and development roadmap.
